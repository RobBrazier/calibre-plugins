# Hardcover.app Calibre Plugin

## Search Flow

1. Search by Hardcover Edition Identifier (`hardcover-edition:###`) - the number
   in the URL when you click on an edition
   > e.g. `8548995` in [https://hardcover.app/books/the-hobbit/editions/8548995]
2. Search by ISBN (`isbn:###`) or ASIN (`mobi-asin:###`) Identifiers
3. Search by Hardcover Book ID (`hardcover-id:###`)
   > e.g. `382700` for [https://hardcover.app/books/the-hobbit]
   > NOTE: the ID is not visible on the website, but useful for reliable identification
4. Search by Hardcover Slug (`hardcover:###` or `hardcover-slug:###`)
   > e.g. `the-hobbit` in [https://hardcover.app/books/the-hobbit/editions/8548995]
5. Search by Title
